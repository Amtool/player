html5版 音乐播放器
======

html5版本音乐播放器，支持iOS设备

#### 功能说明
> * 1、支持iOS设备，但是iOS不支持自动下一曲，这是iOS本身限制，支持touch事件
> * 2、支持播放模式：循环，单曲循环
> * 3、支持歌词实时显示，包括显示到title
> * 4、支持静音，iOS不支持……

#### jQuery版本
withjQuery里面是jquery版本的播放器，之前只是想模仿下亦歌，于是就用jQ写了，后来大家反映可以搞个纯javascript的html5版本，于是出了最新的这个版本
